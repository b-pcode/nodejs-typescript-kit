# NodeJS + Typescript Starter Kit

[![code style: prettier](https://img.shields.io/badge/code_style-prettier-ff69b4.svg?style=flat-square)](https://github.com/prettier/prettier)

## Installing Dependencies

```bash
yarn install
```

## Building the App

```bash
yarn build
```

## Running tests

```bash
yarn test
```

## Linting

```bash
yarn lint
```
